# 🍰 Cake API

API REST desenvolvida em Java com Spring Boot para gerenciamento de bolos e pedidos.  
O projeto foi criado como estudo de desenvolvimento **backend profissional**, utilizando autenticação com **JWT** e banco de dados **PostgreSQL**.

---

# 🚀 Tecnologias utilizadas

- Java 17+
- Spring Boot
- Spring Security
- JWT (JSON Web Token)
- PostgreSQL
- JPA / Hibernate
- Maven
- Insomnia (testes de API)

---

# 📦 Funcionalidades

✔ Cadastro de usuários  
✔ Autenticação com JWT  
✔ Controle de acesso com roles  
✔ CRUD de bolos  
✔ API REST estruturada  
✔ Integração com banco PostgreSQL  

---

# 🔐 Autenticação

A API utiliza **JWT (JSON Web Token)** para autenticação.

Fluxo:

1. Criar usuário
2. Fazer login
3. Receber token
4. Usar token nas requisições protegidas
